## **Introducing Plaza WhatsApp Widget**

A truly plug-and-play WhatsApp button for any website. One script tag. Zero dependencies. 2KB gzipped.

```html
<script src="https://cdn.jsdelivr.net/gh/plazatechnologies/whatsapp-widget@1/dist/whatsapp-widget.min.js"
        data-whatsapp-phone="5511999999999"
        data-whatsapp-message="Hi! I'm interested"></script>
```

That's it. No npm install. No build step. No jQuery. Just paste and go.
